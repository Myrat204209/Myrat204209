# 💫 About Me:
🔭 **I’m currently working on**<br>
A production-grade Smart Home Ecosystem built with Flutter. It features role-based access control (RBAC), multi-property management, and real-time hardware orchestration (Sensors & Controllers). I am heavily focused on building this using a strict, scalable multi-package architecture and high-performance monitoring dashboards.

👯 **I’m looking to collaborate on**<br>
Scalable Micro-SaaS architectures and advanced state management patterns. I’m particularly interested in projects pushing the boundaries of the BLoC pattern within modular Flutter environments.

🤝 **I’m looking for help with**<br>
Optimizing low-latency WebRTC/RTSP camera streaming and efficient playback of high-bitrate security footage within Flutter.

🌱 **I’m currently learning**<br>
I am expanding my stack into the native and backend realms: learning **Golang** for high-performance backend systems, and **Swift (iOS)** & **Kotlin (Android)** for deep native integrations. I'm also continuing to refine automated deployment pipelines for multi-platform apps.

💬 **Ask me about**<br>
Flutter Clean Architecture, building modular multi-package systems (`lib`, `data_provider`, `app_ui`, `form_inputs`), Firebase integration, and why the Command Line is superior for build management.

⚡ **Fun fact**<br>
I’m a firm believer that if you can’t deploy it from a terminal, the workflow isn't finished yet.
